# Code of Conduct

Open Hipp0 adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

- Be respectful. Assume good faith.
- Keep discussion technical. Disagreement about code is normal; personal
  attacks and harassment are not.
- Zero tolerance for harassment, discrimination, doxxing, or NSFW content.
- If someone crosses the line, report it — privately — to the maintainers
  via the contact in [`SECURITY.md`](./SECURITY.md) (same channel).

## Scope

This code applies to project spaces (GitHub issues, PRs, discussions,
Discord/forum when they exist) and to public behavior that represents the
project (conference talks referencing Open Hipp0, social posts tagging
the project).

## Enforcement

Maintainers have the authority to remove comments, close PRs, and ban
accounts for conduct violations. Enforcement actions are private by
default; the reporter is kept informed.
