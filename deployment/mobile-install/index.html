iPhone / iPad TestFlight

TestFlight is Apple's public beta channel. You'll install the TestFlight app, then tap a link to join the Open Hipp0 Remote beta.

Alternative: sideload the signed .ipa
  1. Install AltStore or Sideloadly on your Mac/PC.
  2. Download the signed IPA: openhipp0-latest.ipa.
  3. Drag the IPA into AltStore/Sideloadly, connect your phone, install.
  4. Free Apple IDs need to re-sign every 7 days; paid ($99/yr) accounts get 1 year.

No App Store review gates this channel. Builds ship as soon as we mark them approved on TestFlight.

If Chrome blocks the install
  1. Open Settings → Apps → Special app access → Install unknown apps.
  2. Pick your browser (Chrome, Firefox, etc.) → "Allow from this source".
  3. Reopen the APK file from Downloads.

The APK is signed with a stable release key (fingerprint in deployment/mobile-install/SIGNING.txt). Google Play Protect might warn because we're not on Play — tap "Install anyway."
